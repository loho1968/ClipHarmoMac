## 核心变更

加密方案从 ECDH P-256 密钥交换 → 配对码 HKDF 派生：

```
旧: ECDH handshake (Keychain/AsyKeyGenerator) → shared secret → HKDF → AES-256
新: roomKey "0T976Q" → HKDF-SHA256(salt="ClipSync-RoomKey-v1") → AES-256
```

两端用同一配对码派生相同密钥，无需握手、无需 Keychain。
