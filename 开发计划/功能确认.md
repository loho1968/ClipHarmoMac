# 使用场景

## 家里使用
- [ ] 一个wifi网络
## 办公室使用
- [ ] 手机连接wifi，Mac连接有线，2个网段。
- [ ] 手机连接5G,Mac连接有线，内网+外网
## 其他场景
- [ ] 手机连接5G,Mac连接其他wifi
- [ ] 手机连接5G,Mac连接手机热点
- [ ] 手机连接酒店wifi，Mac连接酒店wifi

# 功能
## 手机和Mac连接
### 家里自动连接，因为都是一个wifi网络，可以用UDP广播发现
### 公司使用之：手机连接wifi，Mac的Ip地址固定，手机端可以记忆下来。后续就自动连接了。
### 其他场景，通过配队码实现连接。
## 手机后台激活
### 手机一直保存后台激活，熄屏后再使用时，主动向Mac发送激活信号，获取最近一次的剪贴板（只需要文字）

# 追求目标
 - [ ] 手机和Mac连接，实现剪贴板同步、发送图片和文件
 - [ ] 尽量无感自动连接和复制剪贴板（只需要文字），特别是公司、家里环境切换。公司Mac周一到周六不关机，建立电脑每天关机。
 - [ ] 连接断了或者掉线，手机自动重连
 - [ ] Mac重启后，自动连接
